## 📘 도서 소개

**정렬·탐색·그래프부터 LSTM·GRU·RNN, 트랜스포머까지!**  
소프트웨어 개발자가 반드시 알아야 할 **핵심 알고리즘 50가지**를
AI 시대에 맞게 한 권에 담았습니다.

전통 알고리즘을 AI·머신러닝 흐름 속에서 함께 이해하도록 구성하여
**이론–원리–응용–실습**을 자연스럽게 연결합니다.

다양한 예제 코드를 통해 바로 실무에 활용할 수 있으며,
개발자로서 알고리즘 역량을 한 단계 업그레이드할 수 있습니다.

---

## ✨ 주요 특징

* **정통 알고리즘 + AI 알고리즘** 융합 설명
* Python 기반 **사례 중심 실습**
* 원리 이해를 위한 **직관적인 그림 정리**
* 코드를 작성하며 따라가는 **Hands-on 학습**
* 최신 AI 흐름 반영(트랜스포머 포함)
